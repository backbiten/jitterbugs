# Safety & Scope Policy

This project is an open-source camera utility focused on **local preview and recording**.

## Allowed
- Camera preview and recording to local storage
- User-controlled start/stop
- Local file management (retain/delete/export)
- Privacy-protecting features (e.g., redaction, blurring, watermarking) that reduce harm

## Not allowed (out of scope)
The project will not implement or accept contributions for:
- Facial recognition / identity matching
- Emotion detection
- Psychological or behavioral inference from video/audio
- "Suspiciousness" scoring or norm-compliance scoring
- Any feature intended to facilitate detainment, custody, restraint, or enforcement actions
- Hidden/background recording intended to monitor people without clear user awareness

## Data handling principles
- Local-first by default
- No cloud uploads by default
- Clear, prominent UI indicators when recording
- No profiling or psychological inference
