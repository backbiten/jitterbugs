# Security Policy

## Reporting a vulnerability
If you discover a security or privacy issue, please report it responsibly.

If GitHub Security Advisories are enabled for this repository, please use them.
Otherwise, open an issue requesting a private channel.

## Privacy expectations
- Local-first behavior by default
- No cloud uploads by default
- Clear indicators while recording
- No profiling or psychological inference (see `docs/SAFETY-SCOPE-POLICY.md`)
